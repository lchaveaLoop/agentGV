# Fiction Writing Skill (小说创作)

## 能力

此技能提供专业的文学创作能力，包括：

- 📖 **故事构思**: 情节设计、世界观构建
- 👥 **角色塑造**: 人物性格、背景故事、成长弧线
- ✍️ **文体风格**: 叙述技巧、对话写作、场景描写
- 🎭 **类型支持**: 科幻、都市、悬疑、言情、历史等

## 使用示例

### 示例 1: 科幻小说
```
用户：写一篇关于时间旅行的科幻小说

Skill 执行:
1. 构建时间旅行规则体系
2. 设计主角和冲突
3. 创造悬念和反转
4. 完成 3000 字短篇故事
```

### 示例 2: 都市小说
```
用户：写一个职场逆袭的都市故事

Skill 执行:
1. 设定职场环境和人物关系
2. 设计冲突和成长线
3. 描写细节和情感
4. 完成励志故事
```

### 示例 3: 故事大纲
```
用户：帮我构思一个悬疑小说大纲

Skill 执行:
1. 设计核心谜团
2. 规划线索和伏笔
3. 安排反转节奏
4. 输出完整大纲
```

## 创作指南

### 1. 故事结构
- **开端**: 引入角色和世界（10-15%）
- **发展**: 建立冲突和关系（30-40%）
- **高潮**: 核心冲突爆发（25-30%）
- **结局**: 解决问题和收尾（15-20%）

### 2. 角色塑造
- 给主角明确的**目标**和**动机**
- 设计合理的**性格缺陷**和**成长空间**
- 配角要有**独立价值**，不只是工具人

### 3. 对话写作
- 符合角色身份和性格
- 推动情节发展
- 避免信息倾泻（info-dumping）

### 4. 场景描写
- 调动五感（视觉、听觉、嗅觉等）
- 服务于情绪和氛围
- 避免过度描写拖慢节奏
