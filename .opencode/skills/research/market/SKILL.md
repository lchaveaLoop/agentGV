# Market Research Skill (市场调研)

## 能力

此技能提供专业的市场调研能力，包括：

- 📈 **行业分析**: 市场规模、增长率、价值链
- 🏢 **竞品分析**: 产品特点、定价策略、市场份额
- 👥 **用户研究**: 目标用户、需求分析、行为模式
- 🔮 **趋势预测**: 技术趋势、市场机会、风险评估
- 📊 **数据分析**: 数据收集、统计分析、可视化

## 使用示例

### 示例 1: 行业分析
```
用户：分析新能源汽车行业的现状和趋势

Skill 执行:
1. 收集行业数据和报告
2. 分析市场规模和增长率
3. 识别关键驱动因素
4. 预测未来发展趋势
5. 输出分析报告
```

### 示例 2: 竞品分析
```
用户：对比分析主要智能手机品牌

Skill 执行:
1. 确定竞品范围
2. 收集产品信息和定价
3. 对比功能和特点
4. 分析市场定位
5. 输出竞品对比矩阵
```

### 示例 3: 用户研究
```
用户：分析 Z 世代的消费习惯

Skill 执行:
1. 定义目标用户群体
2. 收集消费行为数据
3. 识别消费模式和偏好
4. 分析影响因素
5. 输出用户画像
```

## 研究方法

### 1. 数据来源
- 行业报告（艾瑞、易观、IDC 等）
- 公司财报和公告
- 新闻资讯和媒体
- 社交媒体和用户评论
- 政府和协会统计数据

### 2. 分析框架
- **PESTEL**: 政治、经济、社会、技术、环境、法律
- **Porter's Five Forces**: 行业竞争结构
- **SWOT**: 优势、劣势、机会、威胁
- **BCG Matrix**: 产品组合分析

### 3. 报告结构
```markdown
# 报告标题

## 执行摘要
- 核心发现
- 关键结论
- 主要建议

## 市场概况
- 市场规模
- 增长趋势
- 市场细分

## 竞争格局
- 主要玩家
- 市场份额
- 竞争策略

## 趋势与机会
- 技术趋势
- 市场机会
- 风险评估

## 结论与建议
```
