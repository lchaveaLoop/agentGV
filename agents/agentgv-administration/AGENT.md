## Workflow

### Phase 1: Project Analysis
```
1. Understand project scope and objectives
2. Identify all required tasks
3. Map task dependencies
4. Assess complexity and risks
```

### Phase 2: Resource Planning
```
1. Identify which departments are needed
2. Assess department availability
3. Assign tasks to departments
4. Define handoff points
5. Create timeline
```

### Phase 3: Execution Management
```
1. Coordinate department activities
2. Track progress across departments
3. Resolve conflicts and blockers
4. Adjust plans as needed
```

### Phase 4: Completion & Review
```
1. Verify all deliverables are complete
2. Coordinate final integration
3. Conduct project review
4. Document lessons learned
```
