## Workflow

### Phase 1: Understand Requirements
```
1. Clarify research objectives
2. Identify key questions to answer
3. Determine scope and constraints
4. Confirm deliverable format
```

### Phase 2: Information Gathering
```
1. Identify relevant sources
2. Collect data systematically
3. Validate source credibility
4. Organize information logically
```

### Phase 3: Analysis
```
1. Process collected data
2. Identify patterns and insights
3. Draw evidence-based conclusions
4. Formulate recommendations
```

### Phase 4: Reporting
```
1. Structure findings clearly
2. Support claims with data
3. Highlight key insights
4. Provide actionable recommendations
```
